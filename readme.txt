软件基本使用流程：
->填入文件保存位置并输入关键词，关键词规则见https://www.ebi.ac.uk/arrayexpress/help/how_to_search.html#AdvancedSearchExperiment。
->设置下载mode和rest time，点击search，开始下载样本标签数据。
->等待其下载完毕。如果程序意外崩溃，请重启并填入相同信息，调大rest time并继续进行下载。
->点击open file，选择后缀为mode缩写的txt文件打开（如_nml.txt）。
->通过右侧的勾选框进行表格展示信息的控制与筛选。
->对于感兴趣的实验，双击任何它的任何一个单元格将其实验编号加入到下载列表中。再次双击一个相同的实验将其实验编号移除。在下载列表中手动修改任何信息都是无效的。
->点击download按钮，等待相关原始数据的下载。由于某些数据占用空间较大，建议您在设置文件保存位置时选择空间较大的分区。
----------------------------------------------------------------------------
补充特性：
->使用settings-set default file position来选择一个默认目录，这样就不用每次都重新输入文件位置了。
->使用settings-save parameters来保存目前的勾选框状况，以便之后使用。
->mode包含Normal，Cell line focus，Not Sorted和Raw四种。Normal模式会使用输入的关键词和预设的细胞系信息进行筛选，适用于癌症样本的搜索。Cell line focus模式会额外生成一份仅包含具有细胞系信息的样本的文件。Not Sorted模式将不会使用我们预设的与癌症相关的标签，而是使用其本来的标签，因此无法排序，格式会较为杂乱。Raw模式完全不进行筛选，因此可以搜索任何关键词。Not Sorted模式和Raw模式下载的文件暂不支持在本工具中打开，如需将其转换为excel文件请手动复制并粘贴到excel表格中去。注意：Raw模式获得的文件将非常大，需要耗费大量时间进行下载。
->rest time是本工具每下载一个实验相关的数据后暂停的时间。当访问ArrayExpress过于频繁时可能需要将这个数值调大。
->export to excel file可以将已经在工具中打开的表格转换为excel文件。
->可以在cell line library中进行筛选用细胞系的自定义。
